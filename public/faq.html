Frequently Asked Questions

What should I do when I don't see anybody else in the room?

First, make sure you have given permission to the microphone request.
If after a while, you still haven't seen anybody in the room and you think this is not correct, please reload the page.

What browsers are supported?

Currently Chrome, Firefox and Edge are supported for the networked experience.
The VR function works better on some browsers depending on your headset. For example, Microsoft Edge is recommended for WMR headsets

Why is it important to wear headphones / earphones?

When a user is not wearing headphones or earphones, the sounds of the speaker go back into the microphone.
This creates a feedback loop, resulting in a horrible noise for all the other users close in the virtual space.

How do I navigate the space?

Navigation is possible using the keyboard and mouse. While the mouse controls the camera orientation, the keyboard controls the motion.
Use the arrow keys to move, alternatively you can use WASD keys, W (front), S (back), A (left), D(right).
